WooCommerce Product Bundles plugin that allows you to define min/max bundled item quantity constraints.

== Description ==

Need a way to create pick-and-mix boxes with the official WooCommerce Product Bundles extension? This plugin allows you to control the minimum or maximum quantity of bundled products that customers must choose in order to purchase a Product Bundle.

Developers may checkout and contribute to the plugin at its [GitHub Repository](https://github.com/somewherewarm/woocommerce-product-bundles-min-max-items).

**Important**: Before installing this plugin please ensure that you are running the latest versions of both **WooCommerce** and **WooCommerce Product Bundles**.
